# Forum Engine 界面位置和显示逻辑说明

## 📍 Forum Engine 在界面中的位置

### 1. **按钮位置**（左侧控制台区域）

Forum Engine 按钮位于**左侧控制台区域**的按钮组中，具体位置：

```
┌─────────────────────────────────────┐
│  Insight Engine  [按钮]              │
├─────────────────────────────────────┤
│  Media Engine    [按钮]              │
│  [下载报告]                          │
├─────────────────────────────────────┤
│  Query Engine    [按钮]              │
│  [下载报告]                          │
├─────────────────────────────────────┤
│  Forum Engine    [按钮]  ← 在这里！  │
│  [下载日志]                          │
│  [查看日志]                          │
│  [统计]                              │
│  [搜索]                              │
│  [导出]                              │
├─────────────────────────────────────┤
│  Report Engine   [按钮]              │
└─────────────────────────────────────┘
```

### 2. **显示区域**（右侧嵌入页面区域）

当点击 **Forum Engine** 按钮后，会在**右侧的嵌入页面区域**显示论坛聊天界面：

```
┌─────────────────────────────────────┐
│  Forum Engine - 多智能体交流        │  ← 标题
├─────────────────────────────────────┤
│                                     │
│  [论坛聊天消息显示区域]              │  ← forum-container
│  - Insight Engine 发言              │
│  - Media Engine 发言                │
│  - Query Engine 发言                │
│  - Forum Host 发言                  │
│                                     │
└─────────────────────────────────────┘
```

## 🔄 显示逻辑

### 默认状态
- Forum Engine 按钮：**可见**（在左侧按钮组中）
- 论坛聊天界面：**隐藏**（`.forum-container { display: none; }`）

### 点击 Forum Engine 按钮后
1. 按钮变为激活状态（添加 `active` 类）
2. 右侧标题更新为 "Forum Engine - 多智能体交流"
3. 论坛容器显示（添加 `active` 类：`.forum-container.active { display: flex; }`）
4. 隐藏其他内容（iframe、报告容器等）
5. 加载论坛日志到聊天区域

## 🎯 如何查看 Forum Engine

### 方法 1：点击按钮（推荐）
1. 在左侧控制台区域找到 **Forum Engine** 按钮
2. 点击该按钮
3. 右侧会显示论坛聊天界面

### 方法 2：查看日志（模态窗口）
1. 点击 **Forum Engine** 按钮下方的 **"查看日志"** 按钮
2. 会弹出模态窗口显示 forum.log 的原始日志

### 方法 3：使用其他功能按钮
- **统计**：查看统计信息（模态窗口）
- **搜索**：搜索论坛消息（模态窗口）
- **导出**：导出论坛数据
- **下载日志**：下载 forum.log 文件

## ⚠️ 注意事项

1. **Forum Engine 不是 Streamlit 应用**
   - 它不像 Insight/Media/Query Engine 那样显示 iframe
   - 它是一个**纯前端聊天界面**，显示论坛讨论内容

2. **需要点击按钮才能看到**
   - 默认情况下论坛界面是隐藏的
   - 必须点击 **Forum Engine** 按钮才会显示

3. **论坛内容来源**
   - 内容来自 `logs/forum.log` 文件
   - 由 ForumEngine 监控器实时写入
   - 前端通过 SocketIO 或 API 获取并显示

## 🔍 如果看不到 Forum Engine 界面

### 检查清单：
1. ✅ 是否点击了 **Forum Engine** 按钮？
2. ✅ 按钮是否变为激活状态（高亮显示）？
3. ✅ 右侧标题是否变为 "Forum Engine - 多智能体交流"？
4. ✅ 浏览器控制台是否有错误？
5. ✅ 后端服务是否正常运行？
6. ✅ `logs/forum.log` 文件是否存在？

### 调试步骤：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签是否有错误
3. 查看 Network 标签，检查 API 请求是否成功
4. 检查 Elements 标签，查看 `forumContainer` 是否有 `active` 类
