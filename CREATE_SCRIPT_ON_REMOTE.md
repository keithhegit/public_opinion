# 在远程主机上直接创建脚本

## 🚀 最简单的方法

如果你已经在SSH连接到远程主机，可以直接在远程主机上创建脚本文件。

### 步骤

1. **在远程主机上创建脚本文件**:

```bash
sudo nano /tmp/deploy-hk-ubuntu.sh
```

2. **复制粘贴脚本内容**

我会在下面提供完整的脚本内容，你直接复制粘贴到nano中。

3. **保存并退出**

- 按 `Ctrl+X`
- 按 `Y` 确认
- 按 `Enter` 保存

4. **执行脚本**

```bash
sudo bash /tmp/deploy-hk-ubuntu.sh
```
